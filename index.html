Devices

Click Add once per strip. Each strip you add joins the sync group — every control below is sent to all ticked devices at the same time. Close the BanlanX phone app first (each strip allows only one connection).

Controls — broadcast to all synced devices

Wrong colours or dead controls? Each strip in the Devices list has its own protocol, colour-order (RGB/GRB/BGR…) and RGB/RGBW dropdowns — try switching them until colours match. (See the troubleshooting note below.)

Effects — BanlanX names; on SP110E these act as mode numbers

Advanced — send raw hex (to all synced devices)

Bytes are written verbatim to characteristic ffe1 on every synced device. Use this to test commands captured from the official app.
